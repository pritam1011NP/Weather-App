# Weather App 🌦️

A responsive web application that displays real-time weather data and air quality information with dynamic background changes based on weather conditions.

## Features
- Current temperature and weather conditions
- Wind speed/direction visualization
- Air quality index with color bar
- Dynamic background images
- Feels-like temperature, humidity, UV index, and more
- Mobile-friendly design

## Setup

1. **Get API Key**  
- Sign up at [WeatherAPI.com](https://www.weatherapi.com/)
- Replace `ffc85f57af49448b81e92630252801` in `Abahaoya.html` with your key
- TRIAL Ends on 11/Feb/2025
- After expiry of this API key , we have to generate a new API key for free usage. 

3. **Run the app**  
Open `Abahaoya.html` in a web browser

## Usage
1. Enter a city name or location
2. Click "Get Weather"
3. View detailed weather information
4. See background change based on conditions

## API Reference
[WeatherAPI Documentation](https://www.weatherapi.com/docs/)

## Customization
- Add/modify background images in `/images`
- Update CSS classes for different weather conditions
- Modify color scheme in the style section
